Every element is considered a rectangle

Consisting of margin, border, padding and content

content box can be adjusted by width and height

padding box sits around content and can be controlled by padding and related properites padding to add spacing between content and the border

border box wraps the content and padding can be controlled by border and related properites

margin box outter most layer and wraps the content, padding and border sections and can be controlled by margin and related properties. Margin adds space outside the border in all directions
